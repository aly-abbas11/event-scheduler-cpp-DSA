# Event Scheduler — DSA C++ Project

A console-based event scheduler built in C++ that demonstrates 
core Data Structures and Algorithms concepts through a practical 
real world application.

## About
Built during my third semester as part of my Data Structures and 
Algorithms course. This project shows how DSA concepts are not just 
theoretical — they solve real problems. Managing, sorting and searching 
events efficiently is something every real world application does.

## Features
- Add events with title, date and priority
- View all events
- Sort events by priority using Bubble Sort
- Search events by title using Linear Search
- Delete events dynamically
- Get highest priority event instantly
- Statistics showing event breakdown by priority

## DSA Concepts Used
- Vectors as dynamic arrays for event storage
- Bubble Sort algorithm for priority based sorting
- Linear Search for event lookup
- CRUD operations on dynamic data structures

## OOP Concepts Used
- Classes and objects
- Encapsulation of data and methods
- Constructor initialization

## Tech Stack
- Language: C++
- IDE: Dev-C++ / Code::Blocks

## How to Run
1. Clone the repository
2. Open the .cpp file in your IDE
3. Compile and run
